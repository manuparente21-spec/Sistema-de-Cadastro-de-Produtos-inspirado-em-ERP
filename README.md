Este é um sistema simples inspirado no módulo de estoque de um ERP.  
Permite cadastrar produtos, excluir produtos e visualizar um relatório com estoque

Funcionalidades
Cadastrar Produto  
- Nome  
- Categoria  
- Preço  
- Quantidade  
 Excluir Produto  
Remove um produto cadastrado pelo nome.

 Relatório de Produtos  
Exibe todos os produtos mostrando:  
- nome  
- categoria  
- preço  
- quantidade  
E destaca produtos com **estoque baixo (menos de 5 unidades)**.
Sair  
Finaliza o programa.
